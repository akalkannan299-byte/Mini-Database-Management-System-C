# Mini Database Management System in C

## Description
A simple Database Management System developed in C using binary file handling.

## Features
- Insert Record
- Display Records
- Search Record
- Update Record
- Delete Record
- Binary File Storage

## Technologies
- C Programming
- File Handling
- Structures
- Dev-C++

## How to Run
1. Open the project in Dev-C++.
2. Compile and Run.
3. Use the menu to manage student records.
